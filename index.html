Summary of Umbralla Hand

We are a non-profit organization looking to reach out and help people, we want to help people see the value in helping others in need in hope to better our community. We accept donations which we only use to help cloth, feed, and shelter the poor. So please go to our Volunteer/Donations page pick a place you want to volunteer at that's close by or donate to our cause to help better some else. If you want to learn how we were founded or how to get in contact go to the About Us page if you want to learn more about what exactly we want to accomplish please go to the Our Goal page. We hope that you can help us make this world a better place.
